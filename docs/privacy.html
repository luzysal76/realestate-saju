제1조 개인정보 수집 항목 및 목적

부동산 사주 앱(이하 "앱")은 사주 분석 서비스 제공을 위해 다음 정보를 수집합니다.

  • 성명(닉네임) — 사주 결과 화면 표시용
  • 생년월일·성별·태어난 시각 — 사주 계산 핵심 정보
✦ 위 정보는 사주 계산 목적으로만 사용되며, 제3자에게 제공되지 않습니다.

제2조 개인정보 저장 방식

  • 기기 내 저장(오프라인) — 입력 정보는 기본적으로 사용자의 기기(스마트폰·브라우저)에만 저장됩니다.
  • 클라우드 백업(선택) — 사용자가 설정 메뉴에서 Firebase 계정으로 로그인 시, 프로필 데이터가 Firebase Firestore에 암호화된 형태로 백업됩니다.
  • 웹 버전 — 브라우저의 IndexedDB(로컬 스토리지)에 저장되며, 브라우저 캐시 삭제 시 데이터가 초기화될 수 있습니다.

제3조 개인정보 보유 및 파기

  • 기기 내 저장 데이터 — 앱 삭제 또는 프로필 삭제 시 즉시 파기
  • 클라우드 백업 데이터 — 계정 삭제 요청 시 30일 이내 완전 파기
  • 서버에서 사용자의 생년월일 등 개인 정보를 별도 분석·저장하지 않습니다
